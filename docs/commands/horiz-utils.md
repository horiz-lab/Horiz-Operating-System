# horiz-utils (標準ユーティリティ群)

`horiz-utils` は、HorizOS 環境下で日常的に利用される POSIX ライクな基本コマンドを提供する、ゼロ依存のユーティリティコレクションバイナリである。

単一の実行可能ファイルとして提供され、コンパイル時にシンボリックリンクまたは呼び出し元の名前 (`env::current_exe()`) に応じて各サブコマンドの振る舞いを決定する。(BusyBox等に似た設計である)

## 収録されているコマンド

以下のコマンド名で `horiz-utils` を呼び出すと、それぞれの機能が実行される。

### 1. `ls` (ディレクトリ一覧表示)

現在のワーキングディレクトリ (CWD) または引数で指定したパス内のファイルおよびディレクトリの一覧を表示する。

```bash
ls
ls /var/log
```

### 2. `cat` (ファイル内容の出力)

引数として与えられたすべてのファイルパスを順に開き、その内容を標準出力（コンソール）へ結合して表示する。

```bash
cat /etc/hostname
cat /var/log/system.log /var/log/audit.log
```

### 3. `echo` (文字列の出力)

引数として渡されたすべての文字列をスペース区切りで結合し、末尾に改行を付与して標準出力へ書き出す。設定値の確認やシェルスクリプト内のログ出力等に利用される。

```bash
echo "Hello HorizOS World!"
echo $USER
```
